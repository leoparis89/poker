# React poker :spades: :hearts: :clubs: :diamonds:

Just for fun.

Try it out [Here](https://poker.levk.me/).

## Project commands :computer:

Run tests.

`yarn test:watch`

Start front end dev server.

`yarn front:dev`

Start back end compilation.

`yarn back:watch`

Start back end dev server.

`yarn back:dev`

Start storybook.

`yarn storybook`

## Env vars that you will need

- CLIENT_ID
- CLIENT_SECRET
- CALLBACK_URL
- REDIRECT_URL
- POI_APP_GOOGLE_AUTH_URL
- POI_APP_SOCKET_IO_URL
